# HR Manager System – Spring Boot

This project is a Spring Boot application developed to manage HR recruitment processes. It implements a one-to-many relationship where one job requirement is mapped with multiple students. The system performs CRUD operations for managing requirements and candidate details efficiently.

## Technologies Used
- Spring Boot
- Java
- REST API
- Postman (API Testing)
- JPA & Hibernate
- MySQL Database

## Project Structure
- Entity – Defines database tables and relationships
- Repository – Handles database operations
- Service – Contains business logic
- Controller – Manages REST API endpoints

## Features
- Create, Read, Update, Delete operations
- Requirement and student mapping
- RESTful API implementation
- Data management using JPA

## Tools Used
- Spring Tool Suite / VS Code
- Postman
- MySQL

## Purpose
To simplify HR requirement management and maintain structured student recruitment data.
